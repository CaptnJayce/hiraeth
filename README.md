# hiraeth
Hiraeth is defined as profound longing and homesickness.

I chose the name hiraeth because I've never found a text editor I can truly call home, the closest is VSCode but it lacks the TUI, convenience, and control, of Neovim. Neovim is good, but requires a lot of overhead, tinkering, and honestly luck - if your plugin maintainer decides to call it quits one day there's going to be a scramble to find a new maintainer or a fork (looking at you null-ls).

For that reason, this is kept purposefully tiny - the bare minimum of what I need when used in conjunction with lazygit and yazi.

There's also Helix but it lacks vim motions - otherwise that'd be an easy choice. Big fumble imo.\
Zed is another good choice but it lacks the TUI.\
There's also emacs but I just cba with that.

Maybe one day I'll make a text editor of my own. If I did, I'd call it Hiraeth.
