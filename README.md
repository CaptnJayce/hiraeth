# hiraeth

Neovim plugins:

| Plugin | Purpose |
|--------|---------|
| `telescope.nvim` | Fuzzy finder |
| `matugen.nvim` | Material You colours from matugen |
| `which-key.nvim` | Keybind hints |
| `mason.nvim` | LSP/tool installer |
| `mason-lspconfig.nvim` | Mason → lspconfig bridge |
| `nvim-lspconfig` | LSP server defaults |
| `conform.nvim` | Format on save |

Dependencies:

| Plugin | Purpose |
|--------|---------|
| `lazy.nvim` | Plugin manager |
| `plenary.nvim` | Telescope dependency |

Hiraeth is defined as profound longing and homesickness.

I chose the name hiraeth because I've never found a text editor I can truly call home, the closest is VSCode but it lacks the TUI and control of Neovim. Neovim is good, but requires a lot of overhead, tinkering, and worst of all - luck. If your plugin maintainer decides to call it quits one day there's going to be a scramble to find a new maintainer or a fork (looking at you null-ls).

For that reason, this is kept purposefully tiny - the bare minimum of what I need when used in conjunction with lazygit and yazi.

Maybe one day I'll make a text editor of my own. If I do, I'd call it Hiraeth.
